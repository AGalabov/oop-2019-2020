## Оценяване:
* Текущ контрол (ТК)\
Контролно (КП1) – 25 т.\
Проект (П) – 25 т.\
**Оценка (ТК)** = КП1 + П\
* Писмен изпит (ПИ) – 50 т.\
Явяването на писмен изпит **НЕ** е задължително - взимат се точките от ТК)

* Крайна формула: **(ТК + ПИ)/2**

**Скалата за оценяване:**

| Точки           | Оценка        |
| :-------------: |:-------------:|
| 26 т. – 30 т.   | Среден 3      |
| 31 т. – 37 т.   | Добър 4       |
| 38 т. – 44 т.   | Много добър 5 |
| >= 45 т.        | Отличен 6     |
